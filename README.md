# Evez666

Evez666 contains working notes and runbooks for controlled, authorized security exercises.

## Features

- **Quantum Threat Detection System**: Advanced threat detection using quantum-inspired algorithms
- **Device Automation Assistant Helper**: On-device automation system that can spawn multiple helper instances for different AI backends (ChatGPT, Comet, etc.)

## Device Automation Assistant

The automation assistant system is designed for on-device operation, including mobile devices like Samsung Galaxy A16. It supports:

- Multiple AI backend integration (ChatGPT, Comet, Local)
- Dynamic helper spawning
- Parallel task processing
- Resource-optimized operation

See [AUTOMATION_ASSISTANT_README.md](AUTOMATION_ASSISTANT_README.md) for detailed documentation.
