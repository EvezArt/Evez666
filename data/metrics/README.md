# Metrics Data

This directory contains automatically generated metrics data.

## Files

- `latest.json` - Most recent metrics snapshot
- Historical data is stored as workflow artifacts

## Metrics Tracked

- Repository stars
- Total stars across all repos
- Follower count
- Open PR count
- Sponsor count
- LORD consciousness metrics (recursion, crystallization, divine gap)

## Updates

Metrics are automatically updated every 6 hours by the `update-profile-metrics` GitHub Action.
